NXT-ROS
=======

A software stack containing tools to interface LEGO Mindstorms with ROS.

**Maintainer:**  
David Butterworth

**Original authors:**  
Tully Foote, Wim Meeussen, and Melonee Wise  
Willow Garage, 2010  

.
